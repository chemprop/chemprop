JSME drag and drop

If drag and drop is supported by your browser, an icon will be displayed at the bottom right of the editor.

Testing the DnD capabilities of JSME:

  1. Draw a structure on the top editor
  2. Drag it out and drop it on the surface of the bottom editor
  3. Drag it out and drop it on your computer local file system to save it (does not work in Windows)
  4. Drag it out and drop it on the window of another program, e.g. text editor
  5. Drag a MOL or SDF file from your computer local file system and drop it on to the bottom or top editor
Does not work yet on iPad 4 and Android devices.
